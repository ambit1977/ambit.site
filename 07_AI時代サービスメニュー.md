# AMBIT 2026 サービスメニュー追補
## 〜AI・ネットビジネス・情報ビジネスの最新ニーズに合わせて〜

> 02_新規サービス案 を **2026年のニーズ** に合わせて拡張する追補です。
> 既存5領域（Data Governance / Analytics / AdTech / MA・CRM / AI・ML）はそのまま、**「上に乗せる」形** で新メニューを追加します。

---

## 1. なぜ追補が必要か

2025〜2026年の市場変化:

| 変化 | コンサル需要への影響 |
| --- | --- |
| 生成AIが **PoC → 本番展開** フェーズへ | 「導入はしたが業務定着しない」企業が大量発生 → 伴走需要急増 |
| **AIエージェント + RAG + MCP** の普及 | 社内データ／ナレッジの「AIに読める形」への再設計が必須 |
| **Cookieless本格化** + サーバーサイドGTM普及 | 計測実装の高度化、1st Party Data戦略の再構築 |
| **クリエイターエコノミー70兆円規模** へ | YouTube/TikTok/LinkedInなど **動画+クリエイター主導** の集客戦略 |
| 中堅・中小企業のAI導入加速 | 大手ファームが取りこぼす中規模案件が **個人事業主・フリーランス** に流れる |

→ **秋山大志氏の強み（計測・ガバナンス・実装・教育）と完全に噛み合う**。

## 2. 新規サービス5本（2026仕様）

### 🆕 N1. AI業務定着伴走サービス（AI Activation Partner）

**こんな悩みに**
- ChatGPTやGeminiを契約したが、社内で使われていない
- PoCはやったが、業務プロセスに溶け込まない
- 部門ごとに勝手にAIツールを契約し始めて、ガバナンスが崩壊しそう

**提供メニュー**
- AI利用状況のアセスメント（ツール棚卸し・ライセンス把握・実利用率分析）
- 業務プロセスごとのユースケース設計（営業 / マーケ / カスタマーサポート / 経営企画）
- プロンプト集・ナレッジベースの社内整備
- **AIガバナンス規程**（利用ポリシー、情報持出ルール、評価基準）
- 効果測定の計測設計（**ここでAMBITの計測スキルが効く**）

**アウトプット例**
- AI利用ガイドライン
- 部門別ユースケースカタログ
- ROI測定ダッシュボード（GA4/BQ的なノウハウを社内AI活用に適用）

**料金レンジ**
- Audit: ¥60万 / 1か月
- Project: ¥200万〜¥500万 / 3〜6か月
- Advisor: ¥35万/月〜

**差別化軸**: 「AI導入支援」会社は山ほどあるが、**「計測・ガバナンス・本番運用」までできる人は希少**。これがAMBITの圧倒的な強み。

### 🆕 N2. RAG / AIエージェント 実装支援（AI Agent Engineering）

**こんな悩みに**
- 社内ナレッジで動くチャットボットを作りたいが、何から始めればいい？
- RAG構築したが、回答精度が出ない
- MCP対応のエージェントを試したい
- LLMアプリの本番運用（評価・モニタリング・改善ループ）が回せない

**提供メニュー**
- ナレッジ設計（**ガバナンス × データ構造化** の合わせ技）
- ベクトルDB選定・チャンク戦略
- RAGパイプライン構築（LangChain / LlamaIndex / Dify / 独自実装）
- MCP対応エージェント設計
- LLM評価基盤（Evals）と継続改善ループの設計

**アウトプット例**
- ナレッジ設計書 + 構造化済みドキュメントセット
- RAGアプリのMVP（Streamlit / Next.js）
- 評価指標ダッシュボード

**料金レンジ**
- PoC: ¥100万〜
- 本番化: ¥300万〜¥800万

**ターゲット**: 中堅企業の情シス・DX推進部門、AI推進室、スタートアップCTO

---

### 🆕 N3. 1st Party Data 戦略 & Cookieless移行（First-Party Data Studio）

**こんな悩みに**
- 3rd Party Cookie廃止後の広告計測が破綻している
- CDP導入したが「データレイク」になっていて活用できない
- サーバーサイドGTM、コンバージョンAPI、Enhanced Conversionsの違いが社内で噛み合わない
- 同意管理（CMP）と広告計測のバランスが取れない

**提供メニュー**
- 1st Party Data戦略の設計（取得→統合→活用までの全体像）
- サーバーサイドGTM 設計・実装・移行支援
- 各広告プラットフォームのコンバージョン連携（CAPI, Enhanced Conversions, Ads Data Hub）
- CMP（OneTrust / Cookiebot等）の運用ポリシー設計
- 計測アーキテクチャ図の作成と社内浸透

**アウトプット例**
- 1st Party Data戦略ドキュメント
- sGTMコンテナのレビュー＋実装PR
- 同意管理 × 計測のマトリクス
- 移行ロードマップ

**料金レンジ**
- Audit: ¥40万〜
- Project: ¥250万〜¥600万
- Advisor: ¥35万/月〜

**秋山大志氏の強みとの接続**: GTM 100日チャレンジで蓄えた実装知見 × データガバナンス視点 × アドテク経験 = この領域の **最強の個人事業主** の一人。

### 🆕 N4. 計測健康診断 PLUS（GA4 × AI 監査パッケージ）

**こんな悩みに**
- GA4の数値が信用できないが、何が壊れているか分からない
- 広告タグが部署ごとに勝手に増えていて把握できない
- BigQueryに溜まったGA4ローデータを誰も触っていない

**進化のポイント（旧「計測健康診断」のアップグレード版）**
- 従来の手作業監査に加え、**LLMを使った計測実装の自動レビュー** を組み合わせる
- BigQueryのGA4ローデータをAIで異常検知し、レポート化

**提供メニュー**
- 計測実装の自動 + 手動レビュー
- データレイヤ・GTMコンテナのLLM分析
- BQ異常検知レポート
- 修正提案（優先順位 + 工数見積もり）

**料金レンジ**
- ベーシック: **¥15万 / 2週間 / レポート1本**（フック商品）
- プロ: ¥40万 / 4週間 / 修正PRまで

**役割**: 02_新規サービス案の「フック商品」を**AI活用版にアップグレード**。同じ¥10〜15万の入口でも、「AIで自動化」を打ち出すと2026年的に強い。

### 🆕 N5. クリエイター × 企業 マーケティング基盤（Creator-First Marketing）

**こんな悩みに**
- YouTube / TikTok / LinkedInに本気で投資したいが、計測が回らない
- インフルエンサー施策のROIが見えない
- 自社オウンドメディアと外部クリエイターの **同時運用設計** ができない
- ライブコマースの計測・LTV分析が手付かず

**提供メニュー**
- クリエイター起用施策の **計測設計**（アトリビューション、UTM運用、シングルソース統合）
- YouTube/TikTok/LinkedIn の **広告 × オーガニック** 統合計測
- ライブコマースの LTV / リピート分析基盤
- インフルエンサー契約の **ROI 評価フレームワーク**
- 自社クリエイター育成（広報・人事と連携）

**アウトプット例**
- クリエイター × チャネル × KPI のマトリクス
- 計測実装（GTM + sGTM + 各SNS Pixel）
- ROIダッシュボード（Looker Studio / BigQuery）

**料金レンジ**
- Project: ¥200万〜¥500万
- Advisor: ¥30万/月〜

**ターゲット**: D2Cブランド、メディア企業、人事広報（採用ブランディング文脈含む）、教育系（旧ベネッセ周辺の知見が効く）

**位置づけ**: **メインのB2Bコンサル**ではなく、**ブランディング・人脈拡大のサブ商品**。情報発信者・クリエイターとつながることで、N5（クリエイター施策）への送客にもなる。

## 6. ペルソナ追加（C-suiteのAI部署）

旧3ペルソナ（A: マーケ責任者、B: 代理店、C: リクルーター）に加え、2026年は以下を意識:

### ペルソナD: 中堅企業のDX推進室・AI推進室

- 役職: DX推進室長、AI推進担当、CIO直下のプロジェクトマネージャー
- 課題: 大手コンサルファームは予算が合わず、SIerは現場の業務理解が浅い。**「実装まで踏み込む独立した専門家」** を探している
- 意思決定基準:
  - 計測・データ周りの **本物の実装スキル**
  - **発注者の言葉** で説明できること
  - 個人事業主の **柔軟性・スピード**

→ **N1〜N3が直撃する層**。LinkedIn広告とInsights記事で取りに行ける。

### ペルソナE: クリエイター・情報発信者

- 形態: noteクリエイター、YouTuber、Podcast配信者、コミュニティ運営者、企業の広報担当
- 課題: 発信はできるが、計測と収益化の設計が弱い
- 意思決定基準: 個人の規模感に合う料金、伴走の温度

→ **N7とブログ発信が刺さる**。秋山氏自身の "発信する個人事業主" のブランドと相性◎。
