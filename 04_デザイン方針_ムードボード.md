# AMBIT デザイン方針・ムードボード

> 目的: Claude Design に渡して 3〜5案のモックを生成するための、**ブランドの言語化** と **Claude Design用プロンプト** をまとめる。

---

## 1. ブランドキーワード

### 1.1 ありたい印象（Do）

- **Professional but warm** — 専門性は譲らず、しかし「相談しやすい個人」の温度
- **Quiet confidence** — 派手さで殴らず、情報密度と落ち着きで信頼を生む
- **Data-aware** — 数値やグラフが「飾り」ではなく「言語」として機能している
- **Independent** — 代理店でもツールベンダーでもない、独立した第三者

### 1.2 避ける印象（Don't）

- 代理店のリッチなアニメーション過多なサイト
- いわゆる「フリーランスエンジニアのポートフォリオ」テンプレ感
- 自己啓発・情報商材っぽい派手なCTAバナー
- グラデーション全面 / ニューロモーフィズム / 過度なクラフト感

## 2. ビジュアル方針

### 2.1 カラーパレット

```
Primary  (Ink)        #0A0A0B   ほぼ黒。テキスト・ロゴ・主要UI
Surface  (Paper)      #FAFAF7   オフホワイト。背景
Surface2 (Mist)       #F1F1EC   セクション区切り
Border   (Hairline)   #E5E5DF
Accent   (Signal)     #1F6FEB   リンク・CTA。Bluish。クールでデータっぽい
Accent2  (Highlight)  #E8F0FF   Accentの淡色背景
Text-Sub              #5B5B58
Success               #2E7D5B
Warn                  #B5651D
```

- アクセントは **1色のみ**。複数の鮮やか色は使わない。
- グラフ表示時は Accent からのトーン違い（濃淡）でレンジを作る。

### 2.2 タイポグラフィ

- **欧文 見出し**: `Inter` または `Söhne` 系（無ければ `Inter` で代替）
- **欧文 本文**: `Inter`
- **和文**: `Noto Sans JP` / `游ゴシック` をフォールバック
- **数値・コード**: `JetBrains Mono`

サイズ階層:
```
Display  56 / 64 / -2% / Bold
H1       40 / 48 / -1% / Bold
H2       28 / 36 / -1% / SemiBold
H3       20 / 28 /  0  / SemiBold
Body     16 / 28 / 0   / Regular
Small    14 / 22 / 0   / Regular
Mono     14 / 22 / 0   / Regular
```

和欧混植: `font-feature-settings: "palt"` を有効化、欧文と和文間の自動カーニング。

### 2.3 レイアウト

- **12カラムグリッド**、ガター 24px、最大幅 1240px
- 行間と余白を広く取る（"silent space" を価値の表現として使う）
- セクション間の縦余白は最低 96px、PCでは 128px

### 2.4 形状・装飾

- 角丸は `4px` を基本、カードでも `8px` 止まり
- ボーダーは 1px hairline、シャドウは原則使わない（必要な箇所だけ `0 1px 0 rgba(0,0,0,.04)`）
- アイコンは線画（Lucide）、太さ 1.5px

### 2.5 モーション

- フェードイン + 微小なY軸ずらし（8–12px）のみ
- ホバーは色変化と下線、スケール変化は使わない
- ページ遷移は `view-transitions` でクロスフェード

### 2.6 写真・画像

- 代表写真は半身、自然光、背景は無地（プロフィール用に1点撮影推奨）
- それ以外は **写真を最小限**にし、テキストとデータビジュアルで構成する
- 必要なら、**ダッシュボード風の自作スクリーンショット**（ぼかし or ダミーデータ）を装飾として使う

## 4. ロゴ・ワードマーク方針

- 当面は **ワードマーク（AMBIT）のみ** で運用
- 書体: Inter SemiBold、トラッキング +50、すべて大文字
- スペースのバリエーション: `AMBIT` / `A·M·B·I·T`（ファビコンや装飾時）
- 余白規定: ロゴ高さの半分を四方クリアスペースとして確保
- ブランドサインオフ: `AMBIT — Marketing, with governance.`

### 5.2 Hero セクション単体用

```

```

## 6. デザインレビュー時の評価軸

Claude Designから出てきたモックは、以下5軸でレビューする:

| 軸 | チェック項目 |
| --- | --- |
| トーン | 「派手な代理店感」「フリーランス・テンプレ感」が出ていないか |
| 階層 | Hero → サービス → 契約 → ケース → 記事 → CTA の順で読める導線か |
| 余白 | スクロールしていて呼吸ができるか。詰め込みすぎていないか |
| データ表現 | グラフ・図解がブランドの「データの言語化」を体現しているか |
| CTA | 押しすぎず、しかし要所で必ず見える設計か |

3案出して、それぞれをこの表で採点 → 上位案でブラッシュアップ。
