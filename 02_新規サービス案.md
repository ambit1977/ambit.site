# AMBIT 新規サービス案

> 現状の8領域（SEO / 広告運用 / 計測 / MA / DMP / メディア収益化 / ML / DeepLearning）を、**買い手が選びやすい単位**に再パッケージする提案。

---

## 0. サービス構造の考え方

現状: **「やれること」を技術領域で並列に列挙**
→ 課題: 買い手にとって、自分の悩みがどのサービスに該当するか分からない。

リニューアル後: **「悩み × 契約形態」の2軸で再編**

```
              ┌──────────────────────────────────────┐
              │  契約形態 (How)                       │
              ├──────────────────────────────────────┤
              │  単発診断 / プロジェクト / 顧問       │
              └──────────────────────────────────────┘
                              ×
              ┌──────────────────────────────────────┐
              │  悩みドメイン (What)                  │
              ├──────────────────────────────────────┤
              │  ① データガバナンス・基盤             │
              │  ② 計測・アナリティクス               │
              │  ③ 広告運用・アドテク                 │
              │  ④ MA・CRM・グロース                  │
              │  ⑤ AI/MLによる打ち手                  │
              └──────────────────────────────────────┘
```

---

## 1. 主力5サービス（What軸）

### S1. データガバナンス & データ基盤設計

**こんな悩みに**
- GA4・CDP・広告タグが部署ごとにバラバラに動いている
- 同意管理（CMP）と社内ポリシーの整合が取れていない
- 「データドリブン」と言っているが定義が曖昧で、KPIが部署ごとに違う

**提供メニュー**
- データガバナンス規程の起草・社内浸透支援
- 計測ポリシー / ネーミングルール / タグ管理ポリシーの設計
- CMP（OneTrust / Cookiebot等）導入・運用設計
- Privacy by Design ワークショップ

**アウトプット例**
- 計測ポリシードキュメント
- イベント設計書（GA4 / CDP共通）
- ガバナンス運用フロー図、RACI

**料金レンジ（参考）**
- 単発診断: ¥50万〜
- プロジェクト: ¥150万〜¥400万
- 顧問: ¥30万/月〜

---

### S2. WEBアナリティクス実装 & GA4移行支援

**こんな悩みに**
- GA4に移行したがレポートが信用できない
- サーバーサイドGTMを導入したいが、どこから始めればいいか分からない
- BigQueryに溜まったGA4ローデータを活かしきれていない

**提供メニュー**
- GA4 / GTM / sGTM 設計・実装
- BigQuery × Looker Studio ダッシュボード構築
- イベント設計・データレイヤ実装レビュー
- 既存実装の監査（Implementation Audit）

**アウトプット例**
- 計測設計書（measurement plan）
- GTM/sGTMコンテナのレビュー＋修正PR
- BQ SQLテンプレ集、Looker Studioテンプレ

**料金レンジ**
- 実装監査（スポット）: ¥30万〜
- フル実装プロジェクト: ¥200万〜¥600万

---

### S3. 広告運用・アドテク戦略

**こんな悩みに**
- 広告代理店のレポートを鵜呑みにしている状態から脱したい
- Cookieless時代の広告計測・最適化に不安がある
- CDP / DMPと広告配信の連携が機能していない

**提供メニュー**
- 媒体・代理店からの独立した運用監査
- コンバージョンAPI / Enhanced Conversions 等の実装支援
- CDP-広告プラットフォーム連携設計
- 入札・予算配分のロジック設計（MMM補助 / インクリメンタリティテスト）

**料金レンジ**
- 監査: ¥40万〜
- プロジェクト: ¥150万〜
- 顧問: ¥30万/月〜

---

### S4. MA・CRM・グロース設計

**こんな悩みに**
- MAツールを入れたが「メール配信ツール」になっている
- ライフサイクル全体の設計がなく、施策がバラバラ
- 既存顧客のLTV向上に対する打ち手が枯れてきた

**提供メニュー**
- カスタマーライフサイクル / ジャーニー設計
- セグメンテーション・スコアリング設計
- A/B テスト設計、実験基盤のレビュー
- リテンション施策の優先順位付け

**料金レンジ**
- プロジェクト: ¥100万〜
- 顧問: ¥25万/月〜

---

### S5. AI / ML 活用支援（実装まで）

**こんな悩みに**
- 機械学習で何かしたいが、社内にデータサイエンティストがいない
- LLMを業務に組み込みたいが、計測 / ガバナンス観点で詰まっている
- 推薦・予測モデルを試したが、本番運用に乗らない

**提供メニュー**
- マーケ領域でのML PoC（CV予測 / LTV予測 / 解約予測 など）
- LLMアプリ（社内ナレッジ検索、レポート自動生成）導入支援
- ガバナンス観点でのAI利用ポリシー策定

**料金レンジ**
- PoC: ¥80万〜
- 本番化: ¥300万〜

## 2. 契約形態パッケージ（How軸）

### P1. **AMBIT Audit** — 単発診断（スポット）

- 期間: 2〜4週間
- 価格: ¥30万〜¥80万
- 用途: 「今の状態に値段をつけ、優先順位を示す」が成果物。
- 想定購入者: マーケ責任者、代理店PM

### P2. **AMBIT Project** — プロジェクト型

- 期間: 1〜6か月
- 価格: ¥100万〜¥600万
- 用途: 課題が特定済みで、設計〜実装まで一気に走らせたい場合。
- 想定購入者: マーケ責任者、CTO、Head of Data

### P3. **AMBIT Advisor** — 顧問契約（月額）

- 期間: 6か月〜
- 価格: ¥25万〜¥50万/月（稼働時間と相談範囲で可変）
- 用途: 「いつでも相談できる隣の席」「採用面接の同席」「ベンダー選定の伴走」
- 想定購入者: スタートアップ経営者、事業責任者

### P4. **AMBIT Fractional** — フラクショナル（業務委託・席を持つ形態）

- 期間: 3か月〜
- 価格: 稼働日数ベース（週1〜2日）
- 用途: 採用文脈。Head of Data / Marketing 的なロールを期間限定で。
- 想定購入者: リクルーター、CEO、COO

## 3. ティザー的に「目玉商品」を作る

サービス一覧は包括的だが、訪問者の最初の一歩を生む**フック商品**を1つ用意する：

### 🎯 **「データ計測健康診断（GA4 + 広告タグ + CMP）」**

- **¥10万 / 2週間 / レポート1本納品**
- 訪問→申込のハードルを最小化するエントリーパッケージ
- ここから S1〜S3 のプロジェクト化につなげる

> **役割**: 「いきなり顧問は重い」訪問者を、低リスクで顧客化する入口。

## 4. サービスを語る順番（IA設計と接続）

トップページ・サービス一覧では、次の順で見せる：

1. 🎯 健康診断（エントリー、低単価）

3. P1〜P4（契約形態、購買フェーズに応じた選択肢）
